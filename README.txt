
# IA Patrimoniale - Déploiement local

## Étapes pour tout lancer depuis ton PC :

1. Place ton fichier `app_ia_patrimoniale_v16.zip` dans ce dossier.
2. Double-clique sur `lancer_local.bat`.
3. Le projet sera dézippé, initialisé, commité et poussé vers GitHub automatiquement.

Le fichier `lancer_deploiement.py` contient le token GitHub intégré (à ne jamais publier publiquement).
